## üìö About This Book

**Python Programming: From Zero to Hero** is a complete guide to mastering Python, one of the most popular and versatile programming languages in the world. Whether you're a complete beginner or an experienced programmer looking to add Python to your toolkit, this book has something for you.

### What You'll Learn

- **Python Fundamentals**: Variables, data types, operators, strings
- **Control Flow**: Conditionals, loops, break/continue
- **Data Structures**: Lists, tuples, dictionaries, sets
- **Functions**: Definition, parameters, scope, lambda functions
- **Object-Oriented Programming**: Classes, inheritance, polymorphism, encapsulation
- **File Handling**: Reading/writing files, context managers
- **Advanced Topics**: Decorators, generators, iterators, comprehensions
- **Standard Library**: datetime, collections, itertools, pathlib
- **Best Practices**: Testing, debugging, PEP 8, clean code

## üöÄ Getting Started

### Prerequisites

- A computer (Windows, Mac, or Linux)
- Internet connection
- Curiosity and willingness to learn
- No prior programming experience required!

### Reading the Book

#### Online (GitHub Pages)

The book is automatically published to GitHub Pages and can be read online at:

```
https://[your-username].github.io/Python/
```

#### Local Development

To build the book locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/[your-username]/Python.git
   cd Python/python-ebook
   ```

2. **Install Quarto**:
   - Download from [quarto.org](https://quarto.org/docs/get-started/)
   - Or use package manager:
     ```bash
     # macOS
     brew install quarto

     # Ubuntu/Debian
     sudo apt install quarto
     ```

3. **Install Python dependencies** (optional, for running code):
   ```bash
   pip install jupyter matplotlib numpy pandas
   ```

4. **Render the book**:
   ```bash
   quarto render
   ```

5. **Preview the book**:
   ```bash
   quarto preview
   ```

The book will open in your browser at `http://localhost:4200`
